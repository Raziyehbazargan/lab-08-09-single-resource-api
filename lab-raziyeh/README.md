# Vanilla-REST-API
=============

> 

## Getting Started


### Prerequisities

- dependencies: 

```
npm install --save node-uuid sup

```

- devDependencies: 
  
```
npm install -D gulp-eslint gulp-mocha mocha gulp chai

```

## Running

In your root server, type in the command **"node server.js"** in your terminal.
